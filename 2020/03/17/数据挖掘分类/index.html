数据挖掘课程作业——分类

Created2020-03-17|Updated2020-04-05
|Post View:

1.介绍

本文用了决策树模型、k近邻模型、朴素贝叶斯模型、SVM模型、神经网络几种模型对car.data数据集（见[2]）进行了分类。

2.数据集

数据来源于UCI机器学习数据库下载。

Car Evaluation Data Set源自一个简单的分层决策模型，该模型最初是为演示DEX而开发的。

由上图这个数据集中有6个属性值

整个数据集一共1728条数据

不含空缺值

具体数据（片段）如下：

它的六个属性值分别为{buying, maint, doors, persons, lug_boot, safety.}

它的label有四种分别为{unacc, acc, good, vgood}

本次实验目标将六个属性值作为input，实现分类

3.工具介绍

实验使用sklearn包。

本次实验用到了决策树sklearn.tree.DecisionTreeRegressor文档地址

classsklearn.tree.DecisionTreeClassifier(criterion=’gini’, splitter=’best’, max_depth=None, min_samples_split=2, min_samples_leaf=1, min_weight_fraction_leaf=0.0, max_features=None, random_state=None, max_leaf_nodes=None, min_impurity_decrease=0.0, min_impurity_split=None, class_weight=None, presort=’deprecated’, ccp_alpha=0.0)

主要参数：criterion：划分函数包括{“gini”, “entropy”}, default=”gini”

max_depth：预剪枝，控制深度

min_samples_split：预剪枝，控制叶子样本数

k近邻模型sklearn.neighbors.KNeighborsClassifier文档地址

class sklearn.neighbors.KNeighborsClassifier(n_neighbors=5, weights=’uniform’, algorithm=’auto’, leaf_size=30, p=2, metric=’minkowski’, metric_params=None, n_jobs=None, **kwargs)

朴素贝叶斯模型sklearn.naive_bayes.GaussianNB文档地址

class sklearn.naive_bayes.GaussianNB(priors=None, var_smoothing=1e-09)

SVM模型sklearn.svm.SVC文档地址

class sklearn.svm.SVC(C=1.0, kernel=’rbf’, degree=3, gamma=’scale’, coef0=0.0, shrinking=True, probability=False, tol=0.001, cache_size=200, class_weight=None, verbose=False, max_iter=-1, decision_function_shape=’ovr’, break_ties=False, random_state=None)

重要参数：C越小容忍性越大

​ kernel选择核函数

​ gamma：作用于非线性核函数，值越大越复杂， 太大容易过拟合，太小模型划分不准确

神经网络sklearn.neural_network.MLPClassifier文档地址

class sklearn.neural_network.MLPClassifier(hidden_layer_sizes=(100, ), activation=’relu’, solver=’adam’, alpha=0.0001, batch_size=’auto’, learning_rate=’constant’, learning_rate_init=0.001, power_t=0.5, max_iter=200, shuffle=True, random_state=None, tol=0.0001, verbose=False, warm_start=False, momentum=0.9, nesterovs_momentum=True, early_stopping=False, validation_fraction=0.1, beta_1=0.9, beta_2=0.999, epsilon=1e-08, n_iter_no_change=10, max_fun=15000)
