独立级联模型

前面已经介绍过独立级联模型下面就用独立级联模型和贪心算法来实现影响力最大化。

1.数据

数据主要使用karate.gml图中有34个点78条边

具体图形是酱婶的

2.贪心算法

  1. 首先将遍历所有点找出最大影响力的点 加入S
  2. 如果只找一个点就可以结束了
  3. 否则，在剩下的点集中再找一个点并入S使影响力最大
  4. 重复直至点数满足需求

3.运行环境

python 3.7

networkx 2.1

4.代码

5.结果

找到的两个点是34和1，影响力分别为4.2,和6.501，找第一个点耗费2.748678207397461第二个点耗费8.546141624450684秒
